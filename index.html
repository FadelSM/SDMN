FAQ / Q&A

What is this tool for? +
This tool helps you find subdomains associated with a given domain using an API.
How do I use it? +
Enter a domain (e.g., example.com) in the input field and click "Search". The tool will fetch and display the subdomains.
What does the arrow do? +
Clicking the arrow next to a subdomain will open that subdomain in a new tab (e.g., http://subdomain.example.com).
What if no subdomains are found? +
If no subdomains are found, a message will be displayed indicating that no subdomains were found for the given domain.
